### 미션 요약
#### 1. 다리 생성
- `다리 길이`: User에게 입력 받는다.
- `다리 이동가능 경로`: System이 칸마다 0, 1 중 한 값으로 결정 with _랜덤_
  - 랜덤 값이 0 : 아래 칸 이동가능
  - 랜덤 값이 1 : 위 칸 이동가능
#### 2. 플레이어 이동
- `이동 위치`: User에게 입력 받는다. (U or D)
- `이동 결과 반환`: User가 입력한 위치가 
  - 이동 가능하면 O 반환
  - 불가능하면 X 반환
#### 3. 게임 분기
- `다리를 끝까지 건넌다`
  - _종료_ with 게임 성공 + 시도 횟수 출력
- `다리를 건너다 실패한다` 
  - _재시작_ -> 2번으로 이동
  - _종료_ with 게임 실패 + 시도 횟수 출력
